定例会

開催場所：東京大学本郷キャンパス・ZOOMのハイブリット開催

開催頻度：毎週3回（火・木・土）

定例会では運営が事前に指定した発表者が、自分が今取り組んでいるプロジェクトや今まで学んできた内容について発表します。定例会の目的は、サークルに所属している様々な人と分野を問わず交流することです。新しくこのサークルに参加した方は、主に定例会を通して他の会員について知ることができます。また今まで詳しくなかった分野についての発表を聞いて、新しい刺激をもらうことも少なくありません。

勉強会

開催場所：ZOOM

開催頻度：１〜２週間に１回程度

勉強会は定例会ではできなかった、より深い内容を集中的に学び合うための会です。開催頻度や時間は勉強会ごとに異なります。サークルに参加している人は全員勉強会を立ち上げることが可能なので、自分の興味がある勉強会が見つからない時も自分で作ることができます。

現在活動している勉強会

ブロックチェーン勉強会

この勉強会ではブロックチェーンの技術に関する記事などを読んでディスカッションをします。実際にコーディングをするわけではないので、コーディングができない方でも気軽に参加できるのが強みです。さらに実際にDAO（分散型自立組織）を運営してみるというプロジェクトも動いているので、ブロックチェーンに興味のある方は必見です。

webアプリ系開発勉強会

web開発勉強会は、最も初心者向けの勉強会でHTML系の入門から、python flutter/reactによるプロダクト開発ポートフォリオ制作に挑戦してスキルアップしますので、フロントエンド/バックエンドをバランスよく習得できます！実際の開発業務やチーム開発に挑戦する環境があり、「起業をしたい」「インターンに参加したい」方にピッタリです！

機械学習-AI勉強会

Kaggleへのチーム出場、理論学術学習や、実際にコーディングをしてみることを主軸に活動しています！webと同じく、チーム開発ができるのにくわえて、大会に参加する楽しさが味わえます！学術や、実際の開発、チーム開発など裾野は広いので、AI機械学習に関して幅広く習得することができます。

競プロ勉強会

この勉強会では競技プログラミング（略して競プロ）の攻略方法について学び・議論します。競技プログラミングとは、決められた時間以内に与えられた問題を解くことができるプログラムを制作する競技になります。この勉強会ではAtCoderを題材として学んでいるので、自分のプログラミング能力が世界でどの程度通用するのか試したい方は必見の勉強会になります。

交流会

開催場所：都内のカフェ・レストラン等

開催頻度：月１・２回

交流会はオフライン限定のイベントです。交流会の目的は他の会員と実際に会うことによって、より深くお互いを知ることが目的です。今まではレストランで食事をしたり、カフェで合同勉強会、BBQなどを行ってきました。これに関しても主催はあくまで運営側ですが、このようなイベントをやってほしいと要望を出すことも可能なので、ご意見等あればSlackの「運営への質問と意見」で投稿していただければと思います。

プロジェクト

開催場所：ZOOM

開催頻度：プロジェクトにより様々

プロジェクトとは実際にアプリ開発などを行い、リリース・運営までを目的とした活動です。現在もこの新規プロジェクトを募集中なので、Slackの「運営への質問と意見」に投稿していただければと思います。
